Random Password Generator

Overview

The Random Password Generator is a Python-based tool designed to generate secure and random passwords based on user-defined criteria. It allows users to specify the length of the password and select various options such as including uppercase letters, lowercase letters, digits, and special characters.

Methods

generate_password(length, use_uppercase, use_lowercase, use_digits, use_special_chars)

This method generates a random password based on the provided criteria.

Parameters:

  • length (int): The length of the password to be generated.
  • use_uppercase (bool): Flag to include uppercase letters in the password.
  • use_lowercase (bool): Flag to include lowercase letters in the password.
  • use_digits (bool): Flag to include digits in the password.
  • use_special_chars (bool): Flag to include special characters in the password.

Returns:

password (str): The generated random password.

Usage

To use the Random Password Generator, follow these steps:

  1. Run the program: 
    python password_generator.py
  2. Follow the prompts to specify the password length and select character types to include.
