Technical Documentation - Random Password Generator

1. Introduction

The Random Password Generator is a Python-based tool developed to create secure and random passwords based on user-defined criteria.

2. Methods

2.1 generate_password(length, use_uppercase, use_lowercase, use_digits, use_special_chars)

This method generates a random password based on the provided criteria.

Parameters:
length (int): The length of the password to be generated.
use_uppercase (bool): Flag to include uppercase letters in the password.
use_lowercase (bool): Flag to include lowercase letters in the password.
use_digits (bool): Flag to include digits in the password.
use_special_chars (bool): Flag to include special characters in the password.

Returns:
password (str): The generated random password.

3. Usage

To use the Random Password Generator, follow these steps:

  1. Clone the repository: 
    git clone https://github.com/your-username/random-password-generator.git
  2. Navigate to the project directory: 
    cd random-password-generator
  3. Run the program: 
    python password_generator.py
  4. Follow the prompts to specify the password length and select character types to include.
